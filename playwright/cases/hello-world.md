# Hello World: Open Desktop App and Enter API Key (Self-Host Flow)

## Goal

Launch the desktop application and go through the **self-host** onboarding flow (not the managed sign-in flow). Select "Self-host" on the welcome screen, then enter an Anthropic API key.

## Steps

1. Launch the App
2. On the welcome screen, click the "Self-host" button
3. Verify that the API key input is rendered
4. Enter your Anthropic API key into the input field
5. Verify that the key was entered

## Expected

- The app should still be running with an input filled with an API key and the Hatch button enabled and ready to be clicked
