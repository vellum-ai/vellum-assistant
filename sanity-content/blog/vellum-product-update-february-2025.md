## ⚡New Model Support &amp; Workflow Enhancements

💡 It may have been a shorter month, but that didn’t stop us from shipping! We improved how you find models, preview Workflows SDK code, and more.

This month we’re also launching our customer referral program so if you would like to receive a $250 giftcard - sign up to participate!

## 🔨 Key New Features

Workflow SDK Code Preview – If you’re in the beta , you can now preview your Workflow’s code representation directly in the UI!

‍

![__wf_reserved_inherit](https://cdn.sanity.io/images/ghjnhoi4/production/43a98e32ca3dfe0d46f771ca1505b694ed23fc51-1380x218.png)

‍

![__wf_reserved_inherit](https://cdn.sanity.io/images/ghjnhoi4/production/ac37f7fe831d4d2e04258d5dfcdef70a0120b781-3456x1990.png)

Restore Button Moved – Now relocated to the History Cards for easier access when reverting Prompts and Workflows to a prior state. Workflow Node Layout Improvements – Input variables now appear above output types , making Workflows easier to read at a glance. Code Execution, Templating, and Final Output Nodes are all affected by this change. Model Picker Enhancements – Find the right model faster with sorting options (by name, introduction date, or last use). Plus, filter and sort preferences are now saved across sessions .

![__wf_reserved_inherit](https://cdn.sanity.io/images/ghjnhoi4/production/b8d745a74c3f9f9a3f6081d8e5f1ce8850132d2c-3456x1918.png)

🗂 Multiple Chat History Variables – You can now include more than one Chat History variable in Prompts where needed.

⚡ Application-Wide Performance Boosts – We did some big overhauls on our backend resulting in faster page load times across the board.

## 🧠 New Model Support:

✅ GPT-4.5 Preview via OpenAI

✅ Llama 3.2 and Llama 3.3 models via AWS Bedrock (us-west-2)

✅ xAI’s Grok Models (Grok Beta, Grok Vision Beta, Grok 2 1212, and Grok 2 Vision 1212)

✅ Multiple Gemini Models via Vertex AI ( Gemini 2.0 Flash 001 , Flash Lite , Pro , Flash Experimental , and Gemini 1.5 Flash )

✅ DeepSeek V3 via Fireworks AI (More model flexibility for your AI workflows)

✅ OpenAI’s o3-mini (2025-01-31 Snapshot) via Azure (supported for self-hosting)

More choices, better performance—pick the model that fits your needs!
