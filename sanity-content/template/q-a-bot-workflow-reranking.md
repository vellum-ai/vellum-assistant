## Content

This agent is an Q&amp;A chatbot (RAG) that leverages a search engine and a reranking model to provide accurate answers based on internal policy documents. It processes user queries and retrieves relevant policy excerpts, ensuring responses are well-cited and contextually grounded.

‍

### How it Works / How to Build It

MostRecentMessage : This node captures the latest user message from the chat history. SearchNode : It performs a search query against a document index (specifically, "vellum-trust-center-policies-2") using the latest message as the query. It retrieves up to 8 relevant documents based on semantic similarity and keywords. CohereAPIRerankChunks : This node takes the search results and reranks them using the Cohere API, enhancing the relevance of the retrieved documents based on the user's query. FormattedSearchResults : It formats the reranked search results into a structured output, preparing them for the next step. PromptNode : This node generates a final answer by prompting a machine learning model with the question and the formatted policy quotes, ensuring the response includes citations. FinalOutput : It outputs the final answer generated by the PromptNode.

### What You Can Use This For

Answering employee questions about company policies. Providing quick access to compliance information for legal teams. Assisting HR with policy-related inquiries from staff. Supporting customer service representatives with policy clarifications.

### Prerequisites

Vellum account. Document index created in Vellum Cohere API key for reranking functionality.

### How to Set It Up

Create a new workflow in your Vellum account. Add the MostRecentMessage node to capture the latest user input. Connect the MostRecentMessage node to the SearchNode to perform a search based on the captured message. Link the SearchNode to the CohereAPIRerankChunks node to rerank the search results. Connect the CohereAPIRerankChunks to the FormattedSearchResults node to format the output. Link the FormattedSearchResults to the PromptNode to generate a final answer. Finally, connect the PromptNode to the FinalOutput node to display the answer to the user.
