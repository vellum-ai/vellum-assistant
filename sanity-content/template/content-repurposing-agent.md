## Prompt

Create a content repurposing agent that takes a webinar transcript, pulls out the main themes, adds supporting stats from light {{Web research}}, and generates three {{LinkedIn}} posts, one recap draft in {{Notion}}, and pings me in {{Slack}} once done.

## Content

### Why you need it ‍

Webinars are full of insights but most of that value sits unused once the session ends. Turning a transcript into content means summarizing, researching, and producing new formats which takes time. This agent handles all of that automatically. It extracts the main themes from the transcript, finds supporting stats online, and produces three LinkedIn posts, one blog outline, and a Slack summary you can share immediately. ‍

### What you need in Vellum

Text input field for adding the webinar transcript A prompt that extracts themes and insights A research step that pulls supporting stats from the web Formatting for three LinkedIn posts, one blog outline, and a Slack summary A workflow that runs the full process in one execution

## FAQ

### How does the agent turn one transcript into multiple pieces of content?

It extracts themes from the transcript, pulls relevant supporting stats from the web, and formats everything into LinkedIn posts, a blog outline, and a Slack summary with no manual rewriting needed.

### What format should the transcript be in?

You can paste the webinar text directly into the input field or upload it as plain text if your workflow supports it.

### Does the agent write full long form content?

It generates a structured blog outline rather than a full article which you can expand or hand off to another agent for long form drafting.

### Can I customize the tone of the LinkedIn posts?

Yes. You can modify the prompt to set voice, tone, and writing style preferences.

### Does it research new stats or only rely on the transcript?

It performs light web research to add relevant supporting data which gives the final content more authority and depth.
