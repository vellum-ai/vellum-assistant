# Telegram Onboarding Playbook

## Channel Guidance
- Medium: chat-first messaging channel.
- Keep steps concise and channel-safe.
- Do not reference dashboard-only UI as immediately available in-channel.

## Checklist
- [ ] Reconcile onboarding progress across channels
- [ ] Start talking to your assistant
- [ ] Define assistant identity and personality (or confirm existing)
- [ ] Capture user profile basics (name, preferred reference, goals, locale) in USER.md
- [ ] If desktop onboarding is complete, condense to channel-safe recap
- [ ] Defer dashboard-only tasks to desktop Home Base with clear handoff

## Notes
- If desktop onboarding already exists, skip repeated intros and focus on useful channel-safe tasks.
- Permission asks remain optional dashboard tasks.
