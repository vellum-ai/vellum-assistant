# Desktop Onboarding Playbook

## Channel Guidance
- Medium: desktop app with dashboard + docked chat.
- Keep onboarding concrete and action-oriented.
- Use the dashboard for dashboard-capable steps.

## Checklist
- [ ] Reconcile onboarding progress across channels
- [ ] Start talking to your assistant
- [ ] Define assistant identity and personality
- [ ] Capture user profile basics (name, preferred reference, goals, locale) in USER.md
- [ ] Generate or open Home Base app from the pre-built dashboard scaffold
- [ ] Complete `Make it mine` (change look and feel)
- [ ] Complete optional task `Enable voice mode`
- [ ] Complete optional task `Enable computer control`
- [ ] Complete optional task `Try ambient mode`

## Notes
- If this is the first onboarding channel, skip reconciliation and proceed directly.
- Do not ask for microphone or computer-control permissions proactively during hatch + first conversation.
